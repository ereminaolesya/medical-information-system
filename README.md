# Medical Information System (Frontend)

Frontend-приложение медицинской информационной системы, разработанное в рамках задания по продвинутому фронтенду.

## Технологии

* React 18
* TypeScript
* TanStack Query
* React Hook Form + Zod
* Vite
* ESLint + Prettier

## Функциональность

### Аутентификация

* Регистрация врача
* Авторизация (JWT)
* Отображение профиля пользователя

### Пациенты

* Список пациентов с пагинацией
* Фильтрация и сортировка
* Просмотр карты пациента
* Создание нового пациента (модальное окно)

### Осмотры

* Просмотр осмотров пациента
* Создание осмотра
* Поддержка повторных осмотров
* Детали осмотра
* Редактирование (через модальное окно)

### Консультации

* Список консультаций
* Ответы врачей
* Редактирование комментариев

### Отчёты

* Генерация отчёта по осмотрам
* Фильтрация по датам и МКБ-10
* Табличное отображение статистики

## Установка и запуск

```bash
git clone https://github.com/ereminaolesya/medical-information-system.git
cd mis
npm install
npm run dev
```

Приложение будет доступно по адресу:

```
http://localhost:5173
```

---

## Основные страницы

* `/login` — авторизация
* `/registration` — регистрация
* `/profile` — профиль пользователя
* `/patients` — список пациентов
* `/patient/:id` — карта пациента
* `/inspection/create` — создание осмотра
* `/inspection/:id` — детали осмотра
* `/consultations` — консультации
* `/reports` — отчёты
